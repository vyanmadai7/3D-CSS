CSS 3D Carousel

This is a  3D rotating image carousel made using only HTML and CSS.
The images rotate in a circle, creating a smooth 3D effect. It works on all screen sizes and looks modern and clean.

## What This Project Does

Shows 10 car images in a 3D circle

The carousel rotates automatically in a full 360Â° loop

When you hover over an image, it becomes slightly bigger

The bottom section shows a title and small description

Works on mobile, tablet, and desktop

 How to Use It

## Download or clone the project:

git clone https://github.com/vyanmadai7/3D-CSS.git


Open the file index.html in your browser.

Replace the images inside the images folder with your own photos if you want a custom carousel.

## What You Can Change

Rotation speed (faster or slower)

Number of images (add or remove slides)

Distance from the center (3D depth)

Images (your own photos)

Text content (title and description)
